# Electric Shop Web App

A modern and responsive e-commerce web application for browsing electric and electronic products. Built with React.js and Context API, this project provides a clean user interface and efficient state management.

🚀 Features
🛍️ Browse electric products
🔍 Product details
🛒 Add to Cart
❤️ Wishlist (if implemented)
📱 Responsive design
⚡ Fast and smooth user experience
🔄 Global state management using Context API

#  Technologies Used
React.js
Context API
JavaScript (ES6+)
HTML5
Tailwind css
CSS3
React Router DOM
Vite
